설치

https://atom.io/ 에서 Installer 다운로드 후 Install.

Proxy 설정

C:\Users\Jungyu\.atom\.apmrc 파일에 다음 내용 추가.

https-proxy=http://address:port
http-proxy=http://address:port
strict-ssl=false

Install Extends Packages
